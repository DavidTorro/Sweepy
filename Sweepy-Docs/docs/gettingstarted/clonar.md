# Clonar el repo

```bash
git clone https://github.com/tu-usuario/sweepy.git
cd Sweepy
```

## Estructura mínima
```
Sweepy/
├─ SweepyApp/        # app Expo Router
│  ├─ src/app/
│  ├─ src/components/
│  ├─ src/utils/
│  └─ assets/
└─ Docs/Sweepy-Docs/ # Docusaurus
```
