# Preparar el entorno

## Dependencias
- Node.js 20+ y npm
- Expo CLI (global)

## Instalar
```bash
# App móvil
cd SweepyApp
npm install

# Documentación
cd ../Docs/Sweepy-Docs
npm install
```

## Variables de entorno (SweepyApp/.env)
```env
API_BASE_URL=http://localhost:3000
NODE_ENV=development
```

## Ejecutar
```bash
# App móvil
cd SweepyApp
npm start   # i / a / w

# Docs
cd ../Docs/Sweepy-Docs
npm start
```
