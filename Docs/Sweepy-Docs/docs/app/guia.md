# Guía de uso

Sweepy es un marketplace que permite alquilar recursos puntuales, publicar objetos que no utilizas, gestionar disponibilidad y precios, y reservar artículos con calendario en tiempo real.

## Funcionalidades principales

### Marketplace de recursos

- Publicación de artículos o espacios
- Filtros por categoría, precio o disponibilidad
- Búsqueda moderna y clara

### Sistema de reservas

- Disponibilidad en tiempo real
- Calendario integrado
- Confirmaciones y notificaciones

### Para Usuarios

- Perfiles completos
- Historial de reservas
- Valoraciones
- Favoritos

### Para Anfitriones

- Crear y gestionar anuncios
- Control de disponibilidad
- Modificación de precios
- Estadísticas básicas

### Panel de Administración

- Gestión de usuarios
- Moderación de anuncios
- Supervisión de la plataforma
- Estadísticas globales

## Cómo usar la app

### Registro y acceso

- Registro: correo + contraseña → completa perfil
- Login: correo + contraseña o social (Google / Apple en iOS)

### Publicar un recurso

1. Ve a **Crear**
2. Completa datos del artículo o espacio
3. Define precio y disponibilidad
4. Publica y aparecerá en el marketplace

### Alquilar un recurso

1. Explora en **Home**
2. Usa filtros para encontrar lo que necesitas
3. Revisa detalles, disponibilidad y valoraciones
4. Reserva con el calendario integrado
5. Contacta al anfitrión por **Chat** si es necesario
