# Sobre la app

Sweepy es una app móvil de servicios de limpieza enfocada en velocidad, claridad y confianza entre clientes y profesionales.

## Diseño y experiencia
- Interfaz limpia, botones grandes y textos legibles.
- Tema claro/oscuro coherente con la identidad aqua/teal.
- Pocas rutas principales: Home, Crear, Chat, Mis Anuncios, Perfil.
- Feedback inmediato: estados de carga, validaciones y toasts breves.

## Flujo principal
1. Explora en Home y abre una tarjeta.
2. Contacta por Chat y negocia.
3. Publica o gestiona desde Mis Anuncios.

## Identidad visual
- Colores: primario `#00d6cf`, acento `#0b3f55`, versiones oscuras para dark mode.
- Tipografía: peso medio/semibold para legibilidad móvil.
- Componentes reutilizables con vidrio (glass) y sombras suaves.

## Qué incluye
- App Expo Router + NativeWind + TypeScript.
- Catálogo de componentes reutilizables.
- Docs en Docusaurus con modo oscuro y gradientes animados.

## Próximos ajustes sugeridos
- Añadir métricas (tiempos de respuesta, calidad del servicio).
- Mejorar onboarding con checklist inicial.
- Guías rápidas dentro de la app (tooltips contextuales).
